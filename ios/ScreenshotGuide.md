# iOS Screenshot Guide (App Store)

## Required device sizes (minimum)
- **6.9" iPhone** (e.g., iPhone 16 Pro Max)
- **6.5" iPhone** (e.g., iPhone 14 Plus)
- **12.9" iPad**

At least **1 screenshot per device size** is required (more recommended).

## Rules of thumb
- Use the same language per set (one full set in **English**, one full set in **Mongolian** if you plan bilingual metadata).
- No misleading UI, no platform logos, no “beta/test” text.
- Avoid showing real personal data. Use a test account.
- Use consistent status bar (time, battery) if possible.

## 5 most important screens to capture
1. **Home / Explore**: value proposition and navigation
2. **Monks list**: browsing experience
3. **Monk profile**: bio, services, reviews
4. **Booking flow**: selecting service/time + confirmation
5. **Dashboard / Upcoming sessions**: schedule + reminders

## Suggested capture checklist (per device)
- Screenshot 1: Home / Explore
- Screenshot 2: Monks list (filters/search if available)
- Screenshot 3: Monk profile
- Screenshot 4: Booking confirmation / payment step (if applicable)
- Screenshot 5: Dashboard (upcoming session + notification settings)
